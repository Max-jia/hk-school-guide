官津 vs 直資 vs 私立：哪種小學適合你家？一文說清四種學校的區別

📅 2026年7月10日 · 🏷️ 小學擇校 · 入门指南

🏠 一個真實的選擇場景
「官津、直資、私立、國際——到底有什麼區別？」這是每一個來港家庭問的第一句話。
從內地來的家長尤其糊塗：為什麼有些學校免費但要搶校網，有些收費卻不限地址？哪一種最適合你家？五分鐘，幫你搞清楚。

香港 669 所小學，分成官立、資助、直資、私立、國際五類。很多內地來港家長的第一個困惑就是：它們有什麼區別？哪種需要交學費？哪種需要住對校網？一條龍又跟這些有什麼關係？

這篇文章用最直白的方式，幫你在 5 分鐘內搞清楚。

核心差異拆解

1. 官立 vs 資助：都是免費，有什麼不同？

對家長來說，兩者幾乎沒區別——都是免費、都要住對校網、都參加政府統一派位。唯一的差別是辦學主體：官立學校由政府直接管理，資助學校由辦學團體（教會、慈善機構等）管理。但在日常教學和升學機制上，兩者一模一樣。

一句話：官立和資助，對你來說就是「免費 + 必須住對校網」。

2. 直資：半免費、半自主的「混血兒」

直資（Direct Subsidy Scheme）是香港獨有的學校類型。政府補貼一部分，家長交一部分學費。因為收了政府資助，直資學校不能完全自主——必須跟從本地課程框架（不能走純 IB），但可以自行設計教學方法、招生方式和課外活動。

直資最大的吸引力：不限校網，全港招生，學校自主面試選拔。你不需要為了心儀的直資學校而搬家。而且直資學校很多是一條龍——小學部直接連接到中學部，12 年不用換學校。

但競爭也極其激烈。熱門直資如拔萃男書院、聖保羅男女中學附屬小學，每年數千人爭奪百來個學位。

3. 私立：完全自主，但學費最貴

私立學校完全不收政府資助，課程、師資、招生全由學校自主決定。也就是說私立學校的課程靈活度最高——可以走本地課程、國際課程、或兩者結合。

跟直資一樣，私立學校不限校網、自行招生。學費差異巨大，從九龍塘學校約 $40,000/年，到拔萃女小學約 $70,000/年，到漢基國際學校超過 $25 萬/年。

4. 國際學校：出國路線的首選

國際學校走 IB/A-Level/AP 等國際課程，適合將來計劃海外升學的家庭。學費最高（$10-25 萬/年），通常有 debenture（債券）或 capital levy 等額外費用。不參加香港本地中學派位。

你該選哪種？——三個問題幫你決定

Q1：你的預算是多少？
→ $0 → 官立或資助
→ $3-7 萬/年 → 直資
→ $4-15 萬/年 → 私立
→ $10 萬+/年 → 國際學校

Q2：你是否願意為了學校搬家？
→ 願意搬到目標校網 → 官立或資助（省學費，但要住對地方）
→ 不願意/不能搬家 → 直資或私立（不限校網，全港招生）

Q3：你對升中確定性的要求有多高？
→ 必須一條龍/直屬 → 優先看直資和私立（多為一條龍）
→ 可以接受升中派位 → 官立和資助也可（但要確認校網的中學資源）

常見迷思澄清

迷思 1：「資助學校一定比直資差」

錯。喇沙小學、瑪利諾修院學校、聖若瑟小學、嘉諾撒聖瑪利學校——這些全港最頂尖的學校，全部是免費的資助學校。直資的優勢在於課程靈活度和一條龍的便利性，不在於教學品質的絕對高低。

迷思 2：「直資 = 英文教學」

不全對。雖然很多直資學校以英文教學為賣點，但也有直資學校以中文為主要教學語言。每所直資學校的教學語言不同，需要逐一確認。

迷思 3：「私立一定比直資好」

沒有必然關係。私立的優勢在於完全自主（課程、招生、師資不受政府限制），但「貴」不一定等於「適合你家孩子」。香港很多優秀的直資學校（英華、聖保羅男女、港大同學會）在教育品質上不輸給任何私立學校。

一句總結

預算 $0 + 願意搬家 → 官立/資助。
預算 $3-7 萬 + 不想搬家 + 想要一條龍 → 直資。
預算充足 + 追求課程自主 → 私立。
出國路線 → 國際。
