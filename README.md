# 香港升学择校平台（hk-school-guide）

帮助**内地来港家庭**（兼顾香港本地家庭）解决香港升学的信息不对称问题：从 PN 班、幼稚园到小学，覆盖择校、报名、求位信、插班、面试等全流程。

## 目标用户
- **主**：内地来港家庭（新移民 / 优才 / 专才）
- **兼顾**：香港本地家庭

## 商业模式
- **免费引流**：智能择校匹配工具（核心钩子）
- **付费变现**（后续阶段）：学校深度研究报告、AI 文书助手（求位信/插班信）、面试准备等

## 功能范围（第一版 MVP）
1. 幼稚园/PN 择校（独立菜单）
2. 小学择校（独立菜单）
3. 智能匹配：快速 4 项（住址 / 语言 / 学费预算 / 规划路线）出初步榜单 → 补填提升精度
4. 学校库：政府开放数据求「全」 + 名校深度精研求「精」

**范围外（第一版先不做）**：中学阶段、在线支付、AI 文书工具（均进二期路线图）

## 用户视角路线图
| 阶段 | 用户能做到什么 | 验证什么 |
|---|---|---|
| 一（MVP） | 免费在线择校，拿到「适合你家」的学校榜单 | 有没有流量 |
| 二 | 花钱买某校深度报告，付完自动收到 | 有没有人愿意付费 |
| 三 | 用文书助手写求位/插班信、看面试攻略 | 能否提高客单价 |
| 四 | 扩展到中学，覆盖全流程 | 能否规模化 |
