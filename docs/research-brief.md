# 第一轮调研简报（2026-07-08）

## 一、用户痛点（内地来港家庭）
1. **认知偏差**：以为「来港=逃离内卷」，实则香港从幼稚园就开始卷，体系更复杂。
2. **语言三座山**：粤语（孩子短板）、英文（约比内地同龄高 3 个年级）、繁体字。
3. **校网/派位看不懂**：像「学区房」，热门校网竞争惨烈（如 34 网首三志愿成功率约 53%）。
4. **插班黄金窗口**：小三~小四最佳；小五下学期后错过「呈分试」影响升中。多数家长不知有此窗口。
5. **不会写求位信/叩门信**：派位不满意需「叩门」，一封好信是关键 → 付费文书服务刚需。

### 关键产品洞察
- **身份决定选校范围**：无香港身份的跨境儿童被划入「虚拟校网」（北区/元朗/屯门/大埔/东涌）；已来港定居者可选全港。→ 匹配前必须先问「身份/住址」。
- **直资学校 = 内地插班主力赛道**（两文三语、对插班生包容度高）。
- **梯度策略**：家长普遍采用「2 冲刺 + 3 匹配 + 2 保底」。

## 四、样本验证（喇沙小学）— 证明字段可取
- 校网 41 / 资助 / 男校；学费免费；中英并重（含普通话）
- 师生比约 1:1.9，每班约 30 人
- 直属喇沙书院（预留 85% 学额），2025 约 100% 升 Band 1
- 有校车
