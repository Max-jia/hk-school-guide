# 学校数据字段模板 v2（2026-07-08 锁定）

「拿得到度」：🟢 官方全有　🟡 多数有（分批补）　🔴 最有价值但最不齐（需逐校核实/标注来源）

## 幼稚园 / PN
| 字段 | 拿得到度 | 来源 |
|---|---|---|
| 学校名 / 地区 | 🟢 | 开放数据 |
| 类型（半日/全日/长全日） | 🟢 | 开放数据 |
| 是否「免费优质幼稚园计划」 | 🟡 | 概览 |
| 学费 | 🟡 | 概览/官网 |
| 授课语言 | 🟡 | 概览/官网 |
| 宗教 | 🟢 | 开放数据 |
| 是否有直属小学 | 🟡 | 概览/官网 |
| 师生比 | 🟡 | 概览 |
| 校车 | 🟡 | 概览/官网 |
| 报名/面试时间 | 🟡 | 概览/官网 |
| **升学数据（小学去向）** | 🔴 | 多为学校自主披露/第三方，**标注来源** |
| 学校特色 | 🟡 | 概览 |

## 小学
| 字段 | 拿得到度 | 来源 |
|---|---|---|
| 学校名 / 地区 | 🟢 | 开放数据 |
| **校网编号** | 🟢 | 校网界线数据 |
| 办学类型（官立/资助/直资/私立） | 🟢 | 开放数据 |
| 性别（男/女/男女校） | 🟢 | 开放数据 |
| 宗教 | 🟢 | 开放数据 |
| 学费 | 🟡 | 概览/官网 |
| 授课语言 | 🟡 | 概览/官网 |
| 师生比 | 🟡 | 概览 |
| 一条龙 / 直属 / 联系中学 | 🟡 | 概览 |
| 校车 | 🟡 | 概览/官网 |
| 报名/叩门时间 | 🟡 | 概览/官网 |
| **升学数据（升中派位）** | 🔴 | 直属/一条龙官方；「X%升Band1」多为估算，**标注来源+参考** |
| banding 参考 | 🔴 | 非官方，标注「参考·非官方」 |
| 学校特色 | 🟡 | 概览 |

## 匹配对应关系
住址 → 校网 / 地区；语言 → 授课语言；预算 → 学费；规划路线 → 办学类型 + 一条龙。

## 铁规矩
🔴 字段**只写可核实的**，估算一律标注来源并注明「参考·非官方」。
