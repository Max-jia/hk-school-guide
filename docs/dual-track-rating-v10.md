# 港校指南双轨评级体系 v10

> 确定日期：2026-07-15 | 首次应用于：啓思小學

---

## 体系设计

一套框架、两条轨道。同用一个等级标签（S/A+/A/B），根据学校类型走不同评价维度：

| 轨道 | 适用学校 | 占比 |
|------|---------|------|
| **A 轨（传统赛道）** | 官立/资助/直资（DSE为主） | ~80% |
| **B 轨（非传统赛道）** | 私立/国际/IB理念（不走传统DSE） | ~20% |

---

## A 轨：四因子 v9（现行体系，不变）

| 因子 | 权重 | 核心指标 |
|------|------|---------|
| 锚点 | 45% | 升中通路质量（直属/一条龙/联系中学Banding） |
| 共识 | 25% | 六大平台综合曝光与口碑 |
| 竞争 | 10% | 入学难度 |
| 世袭 | 20% | 创校历史/校友网络 |

---

## B 轨：五维评分（新增）

### 一、出口质量（35%）

| 因子 | 权重 | 数据来源 |
|------|------|---------|
| IB/A-Level 均分 | 15% | 校方公布 + SOSOMAMA |
| 海外大学录取 | 10% | 校方公布 + 媒体报道 |
| 英语水平表现 | 5% | IELTS/IGCSE/双语IB文凭率 |
| 中学衔接确定性 | 5% | 教育局/校方（一条龙=5/联系=2/无=0） |

### 二、课程与教学（25%）

| 因子 | 权重 | 数据来源 |
|------|------|---------|
| 课程认证/成熟度 | 8% | IBO/CIS/校方（PYP认证=8/IB理念自研=4/无=0） |
| 语言沉浸深度 | 7% | PSP2025 + 校方（全英=7/双语=5/中文为主=2） |
| 第三语言/特色课程 | 5% | 校方官网 |
| 探究式/项目式学习 | 3% | 校方 + BK（STEAM/脑基础/跨学科） |
| 评估方式 | 2% | PSP2025（无考试/持续评估=2） |

### 三、师资力量（20%）

| 因子 | 权重 | 数据来源 |
|------|------|---------|
| 教师培训率 | 6% | PSP2025 |
| 教师经验稳定性 | 6% | PSP2025（10年以上经验占比） |
| 硕博比例 | 4% | PSP2025 |
| 外籍教师/NET | 4% | PSP2025 |

### 四、校园与资源（10%）

| 因子 | 权重 | 数据来源 |
|------|------|---------|
| 设施丰富度 | 4% | PSP2025（特别室/泳池/STEAM室等） |
| 校车覆盖 | 2% | PSP2025 |
| 课外活动广度 | 2% | 校方官网 |
| SEN/学习支援 | 2% | PSP2025 |

### 五、家长净推荐值（10%）

| 因子 | 权重 | 数据来源 |
|------|------|---------|
| BK/论坛正向率 | 5% | 教育王国 BK 帖综合 |
| 核心痛点反馈 | 3% | BK + 小红书 |

## 原则

1. **数据诚实**：每个因子必须有可核实数据来源，查不到诚实标注「暂无数据」、该因子得 0 分
2. **口碑多源**：家长推荐值相关因子须有 ≥2 个独立来源交叉确认
3. **主观标注**：所有评分标注 💬
4. **非官方声明**：所有评级标注「参考·非官方」
