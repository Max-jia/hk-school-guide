# 香港升学择校平台 · 竞品深度研究报告

> 研究日期：2026-07-15 | 来源：WebSearch 多轮交叉验证

---

## 一、竞品格局总览

香港升学择校赛道分为 **四类玩家**，hk-school-guide 横跨其中多类：

| 类型 | 代表 | 核心模式 | 跟我们的关系 |
|------|------|---------|------------|
| **数据工具型** | Schooland / MySchool / BigExam | 学校数据库 + 搜索筛选 | 直接竞争 |
| **内容媒体型** | Ohpama / GoodSchool | Blog 文章 + 升学资讯 | 直接竞争 |
| **排名权威型** | HKPEP / SOSOMAMA | 排名榜 + 数据对比 | 间接竞争 |
| **论坛社区型** | 教育王国 | UGC 讨论区 | 互补（我们引用其口碑） |

---

## 二、Top 5 竞品深度拆解

### 🥇 Schooland.hk（升学天地）— 最直接的竞品

**网站架构：**
- 学校搜索页：多维筛选器（18区/校网/类型/宗教/性别/Banding）→ 结果列表 → 点击进单校档案
- 单校档案页：地址、电话、网址、学费、师生比、班级结构、教学语言、办学理念、校车、Banding 参考
- 升学锦囊（blog）：选校指南、派位分析、面试攻略、一条龙名单
- **无**：择校匹配工具、深度报告、付费墙

**Blog 内容矩阵：**

| 主题 | 覆盖度 |
|------|--------|
| 小一入学统筹办法（自行+统一派位） | ★★★★ |
| 幼稚园选校指南 + K1 收生时间表 | ★★★ |
| 全港官津小学历年派位数据分析 | ★★★ |
| 直资/私立学费用对比 | ★★ |
| 面试全攻略 | ★★ |

**写作风格：**
- 口吻：中立资讯型，类似政府公告风格，不加主观评价
- 数据：引用教育局官方数据为主，少第三方来源
- 长度：短文为主（500–1000 字）
- 署名：无专家署名，团队统一发布
- SEO：标题关键词覆盖完整（如"2025 小一入学统筹办法"）

**收费模式：完全免费**（无付费墙、无付费报告）

**我们的差异化机会：**
- ✅ 他们有数据，我们有匹配引擎（他们不做的）
- ✅ 他们有短文资讯，我们有 8 章深度报告（他们不做深度）
- ✅ 他们有中立口吻，我们有专家观点和第三方口碑（他们回避的）
- ⚠️ 他们的数据库 UI 和筛选体验可能比我们好（需实测对比）

---

### 🥈 Ohpama（Oh!爸妈·亲子头条）— Blog 内容最强竞品

**背景**：星岛新闻集团旗下亲职数码平台，收录 500+ 香港小学 + 800+ 海外学校资料

**网站架构：**
- 分类目录：小一入学 / 中学 / 幼稚园 / 国际学校 / 海外升学
- 学校比较系统：可对比学校、评估入学机会
- 专家专栏：赵荣德（前喇沙副校长）定期撰稿
- 讲座活动：实体/线上升小讲座

**Blog 内容矩阵（最值得研究）：**

| 内容类型 | 占比估算 | 代表文章 |
|---------|---------|---------|
| 选校策略分析 | ~30% | 《从个案与家长分析两个选小学的策略》 |
| 派位/面试贴士 | ~25% | 直资私立小学面试题目、备试技巧 |
| 政策解读 | ~20% | 教育局最新政策、收生日程 |
| 学校介绍 | ~15% | 名校专题、开放日汇总 |
| 亲子/育儿 | ~10% | 育儿心得、亲子活动 |

**写作风格（🏆 最值得学）：**
- 口吻：专家 + 亲切，类似「资深前辈跟你分享经验」，不冷冰冰
- 赵荣德专栏是核心 IP — 有真人专家背书，信任度极高
- 数据使用：官方数据 + 真实个案结合（既有「教育局说 X%」也有「去年有个家长这样做…」）
- 文章长度：中等偏长（1500–3000 字），不碎片化
- SEO：标题含关键词 + 专家名（如「赵荣德」「升小策略」）
- 更新频率：高，配合教育局收生日程节奏推送

**收费模式：免费**（靠星岛集团广告支撑）

**我们的差异化机会：**
- ✅ Ohpama 有赵荣德，我们可以打造自己的专家 IP（比如「港校指南教育研究团队」）
- ✅ Ohpama 是通用亲子媒体，我们是专做择校深度——更聚焦
- ⚠️ Ohpama 的写作风格比我们「暖」——我们的 blog 偏资讯型，缺人情味和个案故事

---

### 🥉 MySchool.hk — 面试数据型竞品

**核心差异点：**
- 历年面试题目库：幼稚园/小学面试真题汇总（这是独家优势）
- 家长评分系统
- 面试形式标注（小组/个面/游戏）
- 收费详情（学杂费）

**写作风格**：极简资讯型，列表为主，无深度文章

**收费模式：免费**

**我们的差异化机会：**
- ✅ 我们有 blog 深度内容（他们基本没有内容）
- ⚠️ 面试题库是他们独门武器，我们可以考虑补这个板块

---

### 🏅 SOSOMAMA — 排名数据型竞品

**核心功能：**
- DSE 成绩排名（中学）+ 升中派位数据（小学）
- 多源 Banding 对比：BigExam Band、Schooland Band 并列显示
- 收费/免费并列比较
- 2025/2026 招生公告聚合

**写作风格**：纯数据表格式，几乎无文字内容

**收费模式**：基础免费，深度排名数据可能有付费墙（待确认）

---

### 🏅 教育王国（Edu-Kingdom）— 论坛社区型

**规模**：125 万会员，2007 年成立，App 月活高

**核心价值：**
- 家长一手口碑（面试经验、派位结果、校风感受）
- 帖子量极大：小一选校 54,739 帖 / 幼校讨论 106,801 帖

**对我们的价值**：我们报告中引用的 BK 家长口碑，主要源自这里

**竞争关系**：互补而非竞争——他们是 UGC 讨论区，我们是结构化内容

## 三、我们的定位分析

| 维度 | 我们 (hk-school-guide) | 竞品最高水平 |
|------|----------------------|------------|
| 学校数据库 | ✅ 669小学+962幼稚园 | Schooland（类似） |
| 智能匹配引擎 | ✅ **独家优势** | 无人做 |
| 深度择校报告 | ✅ **8章·付费·独家** | 无人做 |
| Blog 内容深度 | 🟡 中等 | Ohpama 更强（专家+个案） |
| 面试题库 | ❌ 没有 | MySchool 独家 |
| 实时排名数据 | 🟡 评级体系（非排名） | SOSOMAMA/HKPEP 更强 |
| UGC 社区 | ❌ 没有 | 教育王国垄断 |
| 写作风格暖度 | 🟡 偏资讯型 | Ohpama 更暖（专家+故事） |

---

## 四、关键结论——我们可以做什么

1. **Blog 内容升级（对标 Ohpama）**：加入「真实个案」叙事风格，打造专家署名 IP，配合教育局收生节奏推文
2. **补一个轻量面试题库**：从 BK/教育王国整理高频面试题，结构化呈现
3. **强化独家护城河**：匹配引擎 + 深度报告应在首页更显眼地展示
4. **不要碰的**：不做 UGC 论坛，不做实时排名
